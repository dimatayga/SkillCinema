# SkillCinema — Приложение-каталог фильмов

SkillCinema — это многофункциональное Android-приложение, разработанное в качестве итогового проекта курса Android-разработки от [Skillbox](https://skillbox.ru/). Приложение предназначено для изучения обширного каталога фильмов, сериалов и актеров. Оно использует API Кинопоиска для предоставления актуальной информации о премьерах, популярных новинках и рейтинговом контенте.

## Основные возможности

- **Главный экран**: Просмотр различных подборок, таких как «Премьеры», «Популярное», «Топ 250», а также случайных наборов по жанрам и странам.
- **Подробная информация о фильме**: Просмотр детальных сведений о фильмах, включая описание, рейтинги, актерский состав и съемочную группу, постеры и похожие фильмы.
- **Профили актеров**: Изучение подробных биографий, фильмографий и наград актеров и режиссеров.
- **Поиск и фильтрация**: Мощный функционал поиска с фильтрами по жанру, стране, году выпуска и рейтингу.
- **Управление коллекциями**: Возможность отмечать фильмы как «Просмотренные», «Любимые», а также добавлять их в пользовательские коллекции и закладки.
- **Локальное кэширование**: Стабильная работа благодаря локальному хранению пользовательских коллекций и статусов просмотра.
- **Медиа-галерея**: Просмотр кадров из фильмов, постеров и фан-артов.

## Технологический стек

Приложение построено с использованием современных инструментов разработки Android и лучших практик:

- **Язык**: [Kotlin](https://kotlinlang.org/)
- **Архитектура**: **MVVM** (Model-View-ViewModel) для четкого разделения ответственности.
- **Внедрение зависимостей**: [Dagger Hilt](https://dagger.dev/hilt/) для надежной и масштабируемой архитектуры DI.
- **Сеть**: [Retrofit](https://square.github.io/retrofit/) и [OkHttp](https://square.github.io/okhttp/) для эффективного взаимодействия с API.
- **Локальная база данных**: [Room Persistence Library](https://developer.android.com/training/data-storage/room) для офлайн-хранения данных.
- **Загрузка изображений**: [Glide](https://github.com/bumptech/glide) для высокопроизводительной загрузки и кэширования изображений.
- **Пагинация**: [Paging 3](https://developer.android.com/topic/libraries/architecture/paging/v3-overview) для плавной загрузки больших списков фильмов.
- **Асинхронная работа**: [Kotlin Coroutines](https://kotlinlang.org/docs/coroutines-overview.html) и [Flow](https://kotlinlang.org/docs/flow.html) для реактивного и неблокирующего программирования.
- **Навигация**: [Jetpack Navigation Component](https://developer.android.com/guide/navigation) для управления переходами между фрагментами.
- **UI**: XML-макеты с использованием **View Binding** и компонентов Material Design.

## Источник данных

Это приложение использует [Kinopoisk Unofficial API](https://kinopoiskapiunofficial.tech/) для получения данных о фильмах.
